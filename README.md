## Overview

### The challenge

Users should be able to:

- View the optimal layout for the site depending on their device's screen size
- See hover states for all interactive elements on the page
- Open a lightbox gallery by clicking on the large product image
- Switch the large product image by clicking on the small thumbnail images
- Add items to the cart
- View the cart and remove items from it

### What I learned

I learnt how to dynamically create `HTML` elements in `js` and update the `DOM` when certain actions are taken. I used this knowledge to create the cart, and product items can be added dynamically by selected the number of items, and clicking the `add to cart` button.
